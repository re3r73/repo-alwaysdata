Comprendre les boucles for en JavaScript

La boucle for permet de répéter une série d'instructions plusieurs fois. Elle est composée de trois parties :

  • Initialisation : définir la valeur de départ du compteur (var i = 0;).
  • Condition : tant que la condition est vraie, la boucle continue (i < saisie;).
  • Incrémentation : à chaque tour, on met à jour la variable (i++; signifie i = i + 1;).

Exemple simple :

for (var i = 0; i < 5; i++) {
    console.log(i);
}

Comprendre les conditions if en JavaScript

La condition if permet d'exécuter un bloc de code uniquement si une certaine condition est remplie.

Exemple de base :

var nombre = 5;
if (nombre > 3) {
    console.log(\"Le nombre est plus grand que 3\");
}

On peut aussi combiner plusieurs conditions avec :

  • && (ET logique)
  • || (OU logique)
  • ! (NON logique)

Comprendre les variables et les opérations arithmétiques en JavaScript

Les variables servent à stocker des données. Pour créer une variable en JavaScript, on utilise var.

Les opérations arithmétiques permettent de faire des calculs simples : + (addition), - (soustraction), etc.

Comprendre les boucles pour les tables de multiplication

Pour réaliser une table de multiplication, on utilise une boucle for pour multiplier une valeur de départ par chaque nombre jusqu'à la limite choisie.

Exemple si l'utilisateur saisit Valeur 1 = 5 et Valeur 2 = 6 :

5 * 1 = 5
5 * 2 = 10
5 * 3 = 15
5 * 4 = 20
5 * 5 = 25
5 * 6 = 30

Ici, on commence à 1 et on répète la multiplication jusqu'à la valeur limite.

    var resultat = "";
